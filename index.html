Rules

  1. The game has four colored buttons: green, red, yellow, and blue.
  2. I will create a sequence of colors that players must repeat by pressing the buttons in the same order.
  3. Each time a player correctly repeats the sequence, I will add another color to the sequence.
  4. If a player makes a mistake and presses the wrong button, the game is over.
  5. Players must repeat the pattern by memory, and cannot rely on a visual cue to help them.
  6. The game gets progressively harder as the number of colors in the sequence increases.
  7. The goal is to get the highest score possible by creating the longest sequence without making a mistake.
